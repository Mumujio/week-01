# 响应式任务感悟
## 一、完成流程
1. 首先复习并使用了css经典的布局：
   1. flex,grid,box    
2. 学习了响应式要用到的知识：
   1. media媒体响应
   2. rem单位（并且使用了flexible.js来根据页面大小调整html的字体大小从而控制rem单位实现响应。）
   3. grid的自动填充 auto-fill属性
   4. 通过js实现点击打开折叠菜单
3. 明确布局思路
   1. 首先菜单和sign Up ，footer区域采用flex结合布局，main区域使用grid自动填充
   2. 采用移动优先的方式，从最开始的375px宽度到1920px逐渐增大的方式逐一设计，并配合媒体响应完成宽度切换时相应      	   css的改变。
   3. 基本样式设计完，最后完成js部分的设计
4. 根据思路完成页面设计
## 二、遇到的难题
1. 不同响应节点时，sign Up区域图片的布局，还有图片的位置
2. 宽度超过1920px时，sign Up区域要实现固定宽度很困难
3. 调整grid自动填充的minmax值
4. js实现点击折叠展开
## 三、未解决问题
